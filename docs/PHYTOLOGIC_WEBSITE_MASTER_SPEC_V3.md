## 2. 理念頁（新增）

### 2.1 頁面定位

理念頁回答的問題是：「植本邏輯憑什麼相信自己做的事是對的？」——這頁講的是品牌的價值判斷標準與更大的時代脈絡，不是個別產品的故事（那是植萃頁的任務）。

### 2.2 內容結構

```
[Section 1] Hero
  └ 大景圖（建議：台灣田野遠景，象徵「土地」與「邏輯」並存的開闔感）
  └ 標語：取自公司簡介「重視生命。尊重自然。相信邏輯。」

[Section 2] 三好
  └ 好喝、好看、好吸收 三點，沿用公司簡介原文精神
  └ 版面：三欄並列或依序捲動呈現，搭配進場動畫（逐句淡入）

[Section 3] 三無
  └ 無人工、無化學、無合成 三點，沿用公司簡介原文精神
  └ 版面風格與「三好」呼應但視覺上需有區隔（如色塊反轉、留白方式不同），避免六點變成一條無差異的列表

[Section 4] 永續飲食計畫
  └ 說明植本邏輯為何堅持台灣在地食材，連結至聯合國 FAO／WHO 於2019年共同定義的「Sustainable Healthy Diets」框架
  └ 核心論點方向（可用於文案擴寫，非逐字稿）：
      - 永續飲食的國際定義強調「低環境衝擊」「保護生物多樣性」「文化適切性」「在地與季節性食材優先」
      - 植本邏輯選用台灣本地食材（地瓜葉、青江菜、薑黃、桂花等），不仰賴長途運輸的進口原料，呼應這個框架中「降低環境足跡」與「支持在地農業經濟」的精神
      - 這不是行銷話術，而是研發起點的真實限制與選擇——公司簡介中「我們走進台灣的鄉間田林與土地，尋找真正有價值的植物來源」這句話，正是這個理念的具體實踐
  └ **內容邊界提醒**：此段落只陳述「為何選擇在地食材」的理念立場，不可延伸出具體的環境數據宣稱（如碳足跡減少百分比），那類數據若無真實第三方認證來源，不可杜撰

[Section 5] 回到首頁 / 前往植萃系列
  └ 輕量導引，連結至植萃頁或首頁
```

## 3. 植萃頁（六頁，結構全面改版）

### 3.1 為什麼要改版

V2 規範將每個系列拆成五個各自接近全螢幕的 Section（開場/心聲/風土/三好三無/回入口），實際執行後出現兩個問題：

1. **體感頁面過多**：每個 Section 接近 100vh，使用者捲動時感覺像在翻過好幾張投影片，而非閱讀一頁完整內容
2. **缺乏產品基本資訊**：整頁從頭到尾都是情感敘事與風土詩意文字，使用者看完仍不知道「這瓶飲料裝了什麼、怎麼喝、喝起來如何」

V3 的解法：**每個系列改為單一頁面、單一滿版大畫面**，原本五段式的內容（心聲、產品資訊）**透過文字動畫疊加在大圖之上分批進場呈現**，而不是切成五張獨立投影片。使用者體感上是「停留在一個畫面，內容隨捲動或時間逐步揭露」，而不是「翻過一頁又一頁」。

### 3.2 單頁結構

```
[單一滿版大畫面，AI生成自然景觀，對應該系列場景方向，貫穿全頁背景不切換]

文字動畫進場內容（依序疊加於大畫面之上，捲動觸發或停留觸發皆可，由 Claude Design 視覺執行決定哪種互動方式更流暢）：

  1. 系列名稱 + 核心句（沿用 V2 第5.2節各色核心句）

  2. 第○個家庭的心聲（沿用 V2 第5.2節各色心聲原文，書信感呈現，不標註歸屬標籤）

  3. ★產品資訊區（新增，純描述性內容，不涉及功效宣稱）：
     - 內含食材：白話列出（如「山藥、老薑、銀耳」，不是成分百分比表）
     - 每份份量：285g
       （這是台灣食品標示法規規定的標準營養標示八大項目，屬於事實性揭露，非功效宣稱，可正常呈現具體數據）
     - 怎麼喝：飲用溫度、建議時機（如清晨空腹或餐前30分鐘溫熱飲用）
     - 口感描述：直接的味覺描述，不是比喻（如「絲滑、微甜帶薑香」），與風土故事的詩意文字區隔開來，這段要讓使用者讀完就知道「喝起來大概是什麼感覺」

  4. 風土語言（沿用 V2 第5.2節各色風土描述文字，但篇幅可比 V2 五段式版本精簡，因為已有產品資訊區承擔「具體資訊」的任務，風土語言只需承擔「氛圍與情感」）

  5. 三好三無呼應（簡短一兩句即可，不需要獨立大標題區塊）

[頁尾固定]：回到植萃系列入口 / 前往下一個系列
```

### 3.4.1 食材清單（ingredients）與風土文字（terroir）的取材原則（V3 新增，重要區分）

實際開發雪山植萃範本與青檸植萃頁面時發現，V2 文件第5.2節原本給的「風土語言方向」只挑了2-4種代表性食材，與實驗室報告書記載的完整配方有落差。這裡正式定義兩種不同性質的內容欄位，取材原則不同，**不可混用同一套清單**：

| 欄位 | 性質 | 取材原則 |
|---|---|---|
| `ingredients`（產品資訊區的食材清單） | 事實性揭露，給使用者知道「裡面裝了什麼」 | **必須完整列出**，不可遺漏，但**這是品牌選擇對外溝通呈現的清單，不等同於法定成分標示**——若未來需要做食品標示法規要求的正式成分標示，須另外核對真實完整配方，不可直接沿用本欄位 |
| `terroir`（風土語言段落） | 情感/感官敘事，營造畫面與香氣聯想 | 可僅挑選最具識別度、最有香氣/色澤特色的1-3種食材重點描寫，不需要也不應該逐一帶到每一種食材，避免文字失焦（青檸植萃曾因平均分配三種食材各寫一句、把黑木耳寫成收尾意象，導致缺乏吸引力，後修正為聚焦芭樂與檸檬的清香酸甜） |

**六色官網呈現用食材清單（已定案，2026/06/30 核對版，與報告書完整配方/V2原始文字皆有出入，請以此表為準）：**
