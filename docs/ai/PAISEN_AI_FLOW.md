# 派森 AI / Dr.Marvin 快篩流程

更新日期：2026-05-20

## 目前定位

目前網站中的快篩模組以 Dr.Marvin 生理狀態快篩呈現，作為派森 AI 健康分析與植物機能飲品推薦的前台入口。

主要檔案：

- `src/components/HealthAssessment.jsx`
- `api/analyze.js`
- `src/components/line/UnlockFullReportCard.jsx`

## 使用者流程

1. 使用者填寫基本資料。
2. 系統依基本資料從題庫選出 7 題。
3. 使用者完成 7 題生理狀態快篩。
4. 系統計算總分、分類分數與產品推薦。
5. 前端呼叫 `/api/analyze` 嘗試產生 AI 分析。
6. 若 AI API 失敗，使用本機 fallback 分析。
7. 產生簡易分析報告。
8. 將報告資料寫入 `assessment_reports`。
9. 顯示 LINE CTA，引導使用者解鎖完整報告。

## 基本資料欄位

- 性別
- 年齡區間
- 身高
- 體重
- 職業 / 工作型態
- 睡眠時間
- 睡眠品質
- 運動習慣
- 飲食型態
- 壓力程度
- BMI 自動計算

## 題庫與計分

目前設定：

- 題庫分類：睡眠、腸胃、壓力、代謝、免疫、肌肉、眼睛
- 每次題數：7 題
- 答案選項：1 到 5 分
- 總分範圍：7 到 35 分

目前分級：

- 13 分以下：狀態穩定
- 14 到 20 分：輕度失衡
- 21 到 27 分：中度警訊
- 28 分以上：高度警訊

## 產品推薦邏輯

目前產品：

- 青檸植萃：腸胃、代謝
- 雪山植萃：睡眠、壓力、免疫
- 玫瑰植萃：女性保養、氣色循環、抗氧化
- 桂香植萃：肌肉、運動恢復、能量代謝
- 紫莓植萃：眼睛、3C 使用、抗氧化

系統會依分類分數計算主推薦與輔助推薦。若性別為女性，玫瑰植萃目前有額外推薦加權。

## AI API

API 路徑：

- `/api/analyze`

環境變數：

- `ANTHROPIC_API_KEY`

目前模型：

- `claude-sonnet-4-20250514`

API 回傳文字後，前端會擷取 JSON 並合併 fallback 結果。若 API 或 JSON 解析失敗，前端仍可用 fallback 完成報告。

## 寫入資料

快篩完成後寫入 `assessment_reports`，包含：

- 基本資料
- BMI
- 作答資料
- 總分與等級
- 分類分數
- 推薦產品
- AI 分析
- 生活建議
- `partial_report`
- `full_report`
- `has_joined_line`

若 Supabase schema 缺少部分欄位，前端目前會嘗試移除缺少欄位後重送，避免整體流程中斷。

## 文案與合規注意

- AI 分析應定位為健康管理與生活型態建議。
- 避免醫療診斷、疾病治療或療效承諾。
- 建議使用「支持」、「幫助」、「維持」、「調節」、「日常保養」等語氣。
- 若使用者有明顯或長期不適，應引導尋求專業協助。
